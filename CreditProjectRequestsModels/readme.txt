CreditProjectRequestsModels

Библеотека классов запросов API сервисов

Структура:

Models - классы DTO
 CreateRequest - запрос создания нового запроса на кредит
 EvaluateRequest - запрос создания скоринг заявки на кредит
 ScoringResultRequest - запрос с результатами скоринга
 StatusRequest - запрос кредитной заявки с результатами скоринга
